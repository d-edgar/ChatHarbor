Report a Bug

Found something that isn't working right? Let us know.

We appreciate you taking the time to report issues. Your feedback helps make ChatHarbor better for everyone. Choose one of the options below to get in touch.

📩 Email

Send a quick email describing the issue. This is the easiest option if you're not on GitHub.

Send Bug Report Email

🛠️ GitHub Issues

Open an issue on our GitHub repository. Great if you want to track progress on the fix or provide detailed technical information.

Open GitHub Issue

What to include

To help us fix the issue quickly, please try to include as much of the following as you can:

  • Your macOS version (e.g., macOS 15.3)
  • ChatHarbor version (found in the About section)
  • Which messaging service was affected
  • Steps to reproduce the issue
  • What you expected to happen vs. what actually happened
  • Screenshots, if applicable

Feature Requests

Have an idea for a new feature or improvement? We'd love to hear it. Feel free to use the same channels above — just change the subject to "Feature Request" so we can keep things organized.
